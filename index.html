Notion de base des réseaux informatiques

RéseauLAN/WANTopologie
▼
1.1 — Définition d'un réseau informatique
Un réseau informatique est un ensemble d'équipements (ordinateurs, imprimantes, serveurs, routeurs…) interconnectés entre eux, capables de communiquer et de partager des ressources (fichiers, imprimantes, connexion Internet, etc.).
💡 Définition clé
Un réseau informatique = plusieurs machines reliées pour échanger des données et partager des ressources.
1.2 — Pourquoi créer un réseau ?
Les principaux avantages d'un réseau :
• Partage de ressources : une seule imprimante pour tous
• Partage de fichiers : accès aux documents depuis n'importe quel poste
• Communication : messagerie interne, appels VoIP
• Sécurité centralisée : gestion des droits d'accès
• Réduction des coûts : moins de matériel nécessaire
1.3 — Types de réseaux par étendue géographique
SigleNom completÉtendueExemple
PANPersonal Area Network~ 10 mBluetooth entre PC et téléphone
LANLocal Area NetworkBâtiment / campusRéseau d'une école
MANMetropolitan Area NetworkVilleRéseau d'une université répartie
WANWide Area NetworkPays / mondialInternet
1.4 — Topologies de réseau
La topologie décrit la manière dont les machines sont reliées entre elles.
TopologieDescriptionAvantageInconvénient
BusToutes les machines sur un seul câblePeu de câblePanne = tout tombe
ÉtoileToutes reliées à un switch centralFacile à gérerSwitch = point critique
AnneauMachines en cercleDébit constantRupture = panne réseau
MailléeChaque machine reliée à plusieurs autresTrès fiableCoût élevé
🌍 Exemple concret

Le réseau de ton lycée utilise une topologie en étoile : tous les ordinateurs de la salle informatique sont branchés à un switch central. Si un PC tombe en panne, les autres continuent à fonctionner normalement.

1.5 — Composants matériels d'un réseau
• Carte réseau (NIC) : interface matérielle d'un ordinateur pour se connecter
• Câble RJ45 (Ethernet) : support de transmission filaire
• Hub : diffuse les données à tous les ports (obsolète)
• Switch (commutateur) : envoie les données uniquement au destinataire ciblé
• Routeur : connecte plusieurs réseaux différents et route les paquets
• Modem : convertit le signal numérique en signal analogique (et inversement)

Modèle OSI et DOD (TCP/IP)

OSIDODCouches réseau
▼
2.1 — Le modèle OSI (7 couches)
Le modèle OSI (Open Systems Interconnection) est un modèle théorique créé par l'ISO pour standardiser les communications réseau. Il divise la communication en 7 couches, chacune ayant un rôle précis.
N°CoucheRôleExemple de protocole
7ApplicationInterface avec l'utilisateurHTTP, FTP, DNS, SMTP
6PrésentationEncodage, chiffrement, compressionSSL/TLS, JPEG, ASCII
5SessionÉtablissement et gestion des sessionsNetBIOS, RPC
4TransportDécoupage des données, fiabilitéTCP, UDP
3RéseauRoutage, adressage IPIP, ICMP, ARP
2Liaison de donnéesAccès au support, adresse MACEthernet, Wi-Fi
1PhysiqueTransmission des bitsCâble RJ45, fibre optique
🧠 Moyen mnémotechnique
Pour retenir les couches OSI de bas en haut : "Physique — Liaison — Réseau — Transport — Session — Présentation — Application"
→ "Pour Les Réseaux Tout S'organise Par Action"
2.2 — Le modèle DOD / TCP-IP (4 couches)
Le modèle DOD (Department Of Defense), aussi appelé modèle TCP/IP, est le modèle pratique utilisé sur Internet. Il regroupe les 7 couches OSI en 4 couches.
Couche DODCorrespondance OSIProtocoles
Application7 + 6 + 5 (OSI)HTTP, FTP, DNS, SMTP, SSH
Transport4 (OSI)TCP, UDP
Internet3 (OSI)IP, ICMP, ARP, RARP
Accès réseau2 + 1 (OSI)Ethernet, Wi-Fi, PPP
🌐 Exemple : Envoyer un e-mail avec Gmail

• Application (Gmail) → utilise SMTP pour composer et envoyer l'e-mail
• Transport (TCP) → découpe l'e-mail en paquets numérotés
• Internet (IP) → chaque paquet est étiqueté avec l'adresse IP destination
• Accès réseau (Ethernet/Wi-Fi) → les paquets sont convertis en signaux électriques ou radio

2.3 — Encapsulation des données
Lors de l'envoi, chaque couche ajoute un en-tête (header) aux données — c'est l'encapsulation. À la réception, chaque couche retire son en-tête — c'est la désencapsulation.
// Encapsulation descendante (émetteur)
Données utilisateur
→ [Segment TCP] : en-tête TCP + données
→ [Paquet IP] : en-tête IP + segment
→ [Trame Eth] : en-tête Ethernet + paquet + FCS
→ [Bits] : 0101110001...

5. Physique : les trames sont converties en bits (0 et 1) envoyés sur le câble ou en Wi-Fi.

💡 Définition
Un protocole de communication est un ensemble de règles et de conventions qui définissent la manière dont les données sont échangées entre deux équipements réseau. Sans protocole, les machines ne pourraient pas se comprendre.

3.3 — Comment fonctionne DNS ?
Le DNS (Domain Name System) est le "bottin téléphonique" d'Internet. Quand vous tapez www.google.com, votre ordinateur ne comprend pas ce nom : il a besoin de l'adresse IP correspondante.
🔍 Étapes de résolution DNS

1. Votre PC demande au serveur DNS local (fourni par votre FAI ou DHCP)
2. Si inconnu, le DNS local demande à un serveur DNS racine (.)
3. Le serveur racine répond : "demande au serveur .com"
4. Le serveur .com répond : "demande au serveur de google.com"
5. Le serveur de google.com répond avec l'IP : 142.250.74.78
6. Votre navigateur utilise cette IP pour se connecter

3.4 — Comment fonctionne DHCP ?
DHCP (Dynamic Host Configuration Protocol) attribue automatiquement une adresse IP à chaque machine qui rejoint le réseau. Processus en 4 étapes : DORA
📋 Processus DORA
Discover → le client cherche un serveur DHCP
Offer → le serveur propose une adresse IP
Request → le client accepte l'offre
Acknowledge → le serveur confirme l'attribution

4.1 — TCP vs UDP
CritèreTCPUDP
ConnexionOrienté connexion (3-way handshake)Sans connexion
Fiabilité✅ Garantit la livraison❌ Pas de garantie
Ordre✅ Données dans l'ordre❌ Ordre non garanti

UtilisationWeb, e-mail, FTPStreaming, VoIP, jeux
🤝 3-Way Handshake TCP
1. Client → Serveur : SYN (je veux me connecter)
2. Serveur → Client : SYN-ACK (ok, je suis prêt)
3. Client → Serveur : ACK (connexion établie)
4.2 — Adressage IPv4
Une adresse IPv4 est composée de 32 bits, écrite en 4 octets séparés par des points. Exemple : 192.168.1.10
Chaque adresse IP a deux parties :
• Partie réseau : identifie le réseau
• Partie hôte : identifie la machine dans le réseau
ClassePlageMasque par défautUsage
A1.0.0.0 – 126.255.255.255255.0.0.0 /8Très grands réseaux
B128.0.0.0 – 191.255.255.255255.255.0.0 /16Moyens réseaux
C192.0.0.0 – 223.255.255.255255.255.255.0 /24Petits réseaux
🔢 Exemple de calcul réseau

IP : 192.168.1.45 — Masque : 255.255.255.0
→ Adresse réseau : 192.168.1.0
→ Adresse de broadcast : 192.168.1.255
→ Plage d'hôtes : 192.168.1.1 à 192.168.1.254 (254 hôtes)

4.3 — Adresses IP privées (RFC 1918)
Ces adresses sont utilisées dans les réseaux locaux et ne sont pas routables sur Internet :
• Classe A privée : 10.0.0.0/8
• Classe B privée : 172.16.0.0/12
• Classe C privée : 192.168.0.0/16

💡 Définition

6.2 — La technologie RAID
💡 RAID
RAID = Redundant Array of Independent Disks : technique qui combine plusieurs disques physiques pour améliorer la performance, la sécurité ou les deux.
NiveauDescriptionAvantageInconvénient
RAID 0Répartition (striping) sur 2 disquesDouble vitesse de lecture/écritureAucune redondance
RAID 1Miroir : copie identique sur 2 disquesSurvie si 1 disque tombe50% de l'espace utilisé
RAID 5Striping + parité (3 disques min.)Bon équilibre perf/sécuritéComplexe à reconstruire
RAID 10RAID 1 + RAID 0 (4 disques)Rapide ET redondantCoût élevé
🏦 Exemple : Serveur d'une banque

Une banque utilise RAID 10 pour ses serveurs : elle a besoin de haute performance (RAID 0) ET de sécurité des données (RAID 1). Si un disque tombe en panne, les données sont intactes sur le miroir.

6.3 — Partitionnement et formatage
Partitionnement : division d'un disque dur en plusieurs zones indépendantes (partitions).

Systèmes de partitionnement :
• MBR (Master Boot Record) : ancien, max 4 partitions primaires, max 2 To
• GPT (GUID Partition Table) : moderne, 128 partitions, disques > 2 To

Systèmes de fichiers (formatage) :
• NTFS : Windows (permissions, journalisation)
• ext4 : Linux (rapide, journalisé)
• FAT32 : compatible tous OS, max 4 Go par fichier
• exFAT : clés USB, cartes SD, grands fichiers

D RAID 10
RAID 10 combine le striping (RAID 0 → performance) et le mirroring (RAID 1 → redondance). C'est le plus robuste mais aussi le plus coûteux.
2. Quel système de fichiers est utilisé par défaut sous Windows ?
A ext4
B FAT32
C NTFS
D exFAT
NTFS (New Technology File System) est le système de fichiers standard de Windows. Il supporte les permissions, la journalisation et les grands fichiers.
✏️ Exercices — Leçon 6
Exercice 6.1

Niveau RAID choisi : RAID 10

13.1 — Définition et principe du CPL
💡 CPL = Courant Porteur en Ligne
Le CPL (Courant Porteur en Ligne), aussi appelé PLC (Power Line Communication) en anglais, est une technologie réseau qui utilise le réseau électrique existant pour transmettre des données numériques.
Le principe est simple : superposer un signal numérique haute fréquence au signal électrique basse fréquence (50 Hz) qui alimente normalement les appareils électriques. Un adaptateur CPL branché dans une prise électrique détecte ce signal et en extrait les données réseau.
13.2 — Comment fonctionne le CPL ?
🔌 Fonctionnement étape par étape

1. Émission : L'adaptateur CPL n°1 (branché au routeur via câble RJ45) reçoit les données réseau et les convertit en signal haute fréquence (1–30 MHz)
2. Transmission : Ce signal voyage sur le câble électrique de la maison/bâtiment
3. Réception : L'adaptateur CPL n°2 (dans une autre pièce) capte le signal haute fréquence
4. Conversion : Il convertit ce signal en données réseau et les envoie via câble RJ45 au PC

• Stable et moins sensible aux interférences que le Wi-Fi
• Bonne portée (jusqu'à 300 m en théorie)

• Ne traverse pas les compteurs électriques différents
• Les multiprises parasites (filtres) bloquent le signal
• Débit réel souvent inférieur au débit théorique
• Perturbations possibles par les appareils électriques

802.11ac est Wi-Fi 5 (5ème génération), opère sur la bande 5 GHz avec un débit max de 3.5 Gbps. Wi-Fi 6 = 802.11ax.

A WEP
B WPA
C WPA2
D WPA3

Exercice 14.1
Un responsable informatique doit choisir entre déployer du Wi-Fi 5 (802.11ac) ou du Wi-Fi 6 (802.11ax) dans une école de 500 élèves utilisant simultanément des tablettes. Comparez les deux normes et conseillez-le.
Comparaison :

Wi-Fi 5 (802.11ac) : Bande 5 GHz uniquement, débit max 3.5 Gbps, bonne performance mais peut saturer avec beaucoup d'appareils simultanés.

Wi-Fi 6 (802.11ax) : Bandes 2.4/5/6 GHz, débit max 9.6 Gbps, technologie OFDMA qui gère mieux les connexions simultanées multiples, meilleure efficacité énergétique pour les appareils.

Conseil : Wi-Fi 6 est fortement recommandé dans ce contexte. Avec 500 élèves connectés simultanément, la technologie OFDMA du Wi-Fi 6 permet de servir plusieurs appareils en même temps sans dégradation notable des performances. C'est spécifiquement pour ce scénario (forte densité d'appareils) que Wi-Fi 6 a été conçu.
Exercice 14.2

3. Créer un réseau invité séparé (VLAN) pour les visiteurs, isolé du réseau interne.
4. Mettre à jour régulièrement le firmware du routeur/point d'accès pour corriger les failles de sécurité.

Transistors bipolaires

BJTNPN/PNPAmplification
▼
1.1 — Définition et structure
Le transistor bipolaire (BJT — Bipolar Junction Transistor) est un composant semiconducteur à trois électrodes : Base (B), Collecteur (C) et Émetteur (E). Il existe en deux types : NPN et PNP.
💡 Définition clé
Le transistor bipolaire est un composant qui permet de contrôler un courant fort (IC) à partir d'un courant faible (IB). C'est le principe de l'amplification.

PNPP–N–PConducteur si IB < 0 (courant sortant de B)Flèche entrante sur E
1.2 — Régimes de fonctionnement
Le transistor BJT peut fonctionner dans 3 régimes :

📐 Formules fondamentales
IC = β × IB  |  IE = IB + IC  |  β = IC / IB
1.3 — Applications
• Amplificateur de signal : son audio, radiofréquences
• Commutateur électronique : allumer/éteindre une charge
• Régulateur de tension : alimentation stabilisée
• Oscillateur : génération de signaux périodiques
⚡ Exemple concret

Un transistor NPN 2N2222 avec β = 100 et IB = 0,5 mA donne : IC = 100 × 0,5 = 50 mA. Ce courant peut alimenter une LED ou un petit relais, commandés par un signal logique 5V de microcontrôleur.

Les trois électrodes du BJT sont : Base (B), Collecteur (C) et Émetteur (E). Anode/Cathode sont pour la diode, Source/Drain/Grille pour le transistor MOS.

IC = β × IB = 200 × 0,1 mA = 20 mA. Le gain β amplifie le courant de base.

Le régime actif (linéaire) est utilisé pour l'amplification car IC = β×IB. Le régime saturé/bloqué est utilisé pour la commutation (interrupteur).

Transistor MOS et MOS FET

MOSFETN-MOSP-MOSCommutation
▼
2.1 — Définition et structure du MOSFET
Le MOSFET (Metal Oxide Semiconductor Field Effect Transistor) est un transistor à effet de champ. Ses trois électrodes sont : Grille (G), Drain (D) et Source (S). Il est commandé en tension (contrairement au BJT commandé en courant).
💡 Principe clé
Le MOSFET contrôle le courant ID (Drain-Source) par la tension VGS appliquée sur la Grille. Courant de grille ≈ 0A → très haute impédance d'entrée.
ParamètreBJTMOSFET
CommandeCourant (IB)Tension (VGS)
Impédance entréeFaibleTrès élevée (MΩ)
Vitesse commutationMoyenneTrès rapide
Pertes en commutationPlus élevéesFaibles
Applications typiquesAmplification audioElectronique de puissance
2.2 — Types de MOSFET
Il existe deux grandes familles :
• N-MOSFET (canal N) : conducteur si VGS > VGS(th) > 0. Le courant va du Drain vers la Source.
• P-MOSFET (canal P) : conducteur si VGS < VGS(th) < 0. Utilisé en commutation côté haut.
Et deux modes de fonctionnement :
• Enhancement (enrichissement) : normalement bloqué, s'ouvre avec VGS. Le plus courant.
• Depletion (appauvrissement) : normalement conducteur, se bloque avec VGS.
2.3 — Applications du MOSFET
• Alimentation à découpage (SMPS) : hacheurs, convertisseurs DC/DC
• Variateurs de vitesse : moteurs électriques
• Onduleurs : convertisseur DC → AC
• Amplificateurs de puissance audio : classe D
• Circuits intégrés CMOS : portes logiques, microprocesseurs
⚡ Exemple concret

Un MOSFET IRF540 (N-channel) peut commuter jusqu'à 33A sous 100V avec une résistance RON = 0,044 Ω. Il est utilisé dans les variateurs de moteurs DC et les alimentations à découpage d'ordinateurs.

Le MOSFET est commandé en tension (VGS). Contrairement au BJT (commandé en courant), le courant de grille du MOSFET est quasi nul.
3. Quel avantage majeur a le MOSFET sur le BJT en électronique de puissance ?
A Il amplifie mieux les signaux audio
B Il coûte moins cher
C Commutation plus rapide et pertes plus faibles

Le MOSFET commute beaucoup plus rapidement que le BJT et présente de faibles pertes en commutation, ce qui le rend idéal pour les alimentations à découpage et variateurs.

Le TRIAC — DIAC

TRIACDIACThyristorAC
▼
4.1 — Le DIAC
Le DIAC (DIode for Alternating Current) est un composant bidirectionnel à deux électrodes (pas de gâchette). Il conduit dans les deux sens dès que la tension appliquée dépasse une tension de seuil (VBO, typiquement ±30V).
💡 Rôle du DIAC
Le DIAC est principalement utilisé comme déclencheur du TRIAC. Il génère une impulsion quand la tension dépasse son seuil, ce qui déclenche la conduction du TRIAC.
4.2 — Le TRIAC
Le TRIAC (TRIode for Alternating Current) est un composant bidirectionnel à trois électrodes : A1 (MT1), A2 (MT2) et Gâchette (G). Il conduit dans les deux sens quand une impulsion est appliquée sur la gâchette.

4.3 — Applications TRIAC-DIAC
• Variateur de lumière (dimmer) : régler l'intensité lumineuse d'une ampoule
• Régulateur de puissance : chauffage électrique, four
• Variateur de vitesse AC : moteurs à courant alternatif
• Gradateur : contrôle par angle de phase
⚡ Exemple concret

Dans un variateur de lumière (dimmer), le DIAC déclenche le TRIAC à un angle de phase précis du secteur 220V. Plus l'angle est grand, moins le TRIAC conduit longtemps, moins la lampe reçoit d'énergie → la lumière est plus faible.

Le couple DIAC+TRIAC est classiquement utilisé dans les variateurs de lumière (dimmers) pour contrôler la puissance délivrée à une lampe par découpe de l'onde secteur.

✏️ Exercices — Leçon 4
Exercice 4.1
Expliquez le principe de fonctionnement d'un variateur de lumière utilisant un DIAC et un TRIAC. Décrivez les étapes du cycle de commande.
Principe (commande par angle de phase) :

1. Le secteur 220V AC alimente le circuit. Un potentiomètre (résistance variable) et un condensateur forment un réseau RC.
2. Le condensateur se charge via le potentiomètre. La vitesse de charge dépend de la valeur R du potentiomètre.

5. En augmentant R, le condensateur se charge plus lentement → le TRIAC se déclenche plus tard dans le cycle → moins d'énergie transmise à la lampe → lumière plus faible.

Transistor unijonction (TUJ)

UJTRelaxationDéclencheur
▼
5.1 — Structure et fonctionnement
Le TUJ (Transistor Unijonction), ou UJT (Unijunction Transistor), est un composant à trois électrodes : Émetteur (E), Base 1 (B1) et Base 2 (B2). Il possède une seule jonction PN, contrairement au BJT.
💡 Caractéristique clé
Le TUJ présente une résistance négative dans sa zone de fonctionnement. Quand la tension en E dépasse un seuil (tension de crête VP), le courant d'émetteur augmente et la tension diminue → le TUJ se bloque ou conduit brusquement.
5.2 — Applications du TUJ
• Oscillateur de relaxation : génère des impulsions de déclenchement
• Déclencheur de thyristor / TRIAC : impulsion précise sur la gâchette
• Générateur de dents de scie : signal triangulaire pour oscilloscope
• Minuterie et temporisateur : circuits de retard simples
📐 Tension de crête
VP = η × VBB + VD    (η : rapport intrinsèque ≈ 0,5 à 0,8)
🧠 Quiz — Leçon 5
1. Combien de jonctions PN possède un TUJ ?
A Une seule jonction
B Deux jonctions
C Trois jonctions
D Aucune jonction
Le TUJ (Transistor Unijonction) possède une SEULE jonction PN, d'où son nom. Le BJT en possède deux.
2. Quelle est la propriété électrique particulière du TUJ ?
A Résistance infinie
B Gain en courant très élevé
C Résistance négative dans sa zone de fonctionnement
D Capacité variable
Le TUJ présente une résistance négative dans sa zone active : quand VE augmente au-delà de VP, le courant augmente mais la tension diminue. Cette propriété permet les oscillations de relaxation.
3. Pour quelle application le TUJ est-il particulièrement utilisé ?
A Amplification haute fréquence
B Oscillateur de relaxation et déclenchement de thyristor
C Stockage d'énergie
D Conversion analogique-numérique
Le TUJ est principalement utilisé comme oscillateur de relaxation (génère des impulsions) et pour déclencher des thyristors ou TRIAC avec une impulsion précise.
✏️ Exercices — Leçon 5
Exercice 5.1
Un TUJ a un rapport intrinsèque η = 0,65 et est alimenté sous VBB = 12V. La tension de jonction VD = 0,7V. Calculez la tension de crête VP.
Formule : VP = η × VBB + VD

Application numérique :

Étude d'un AOP en régime linéaire

AOPRégime linéaireAmplificateur
▼
6.1 — Définition et symbole
L'AOP (Amplificateur Opérationnel) est un circuit intégré différentiel à très haute gain. Il possède :
• Entrée non-inverseuse (+)
• Entrée inverseuse (−)
• Une sortie
• Deux alimentations : +Vcc et −Vcc (symétriques)
💡 AOP idéal
• Gain différentiel Ad = ∞  |  Impédance d'entrée Ze = ∞  |  Impédance de sortie Zs = 0  |  Bande passante = ∞
6.2 — Règles de l'AOP en régime linéaire
En régime linéaire (avec contre-réaction négative) :
• La tension différentielle est nulle : V+ = V−
• Les courants d'entrée sont nuls : I+ = I− = 0
📐 Montages fondamentaux
Inverseur : Vs = −(R2/R1) × Ve  |  Non-inverseur : Vs = (1 + R2/R1) × Ve

VT+ = +(R1/R2) × Vsat  |  VT− = −(R1/R2) × Vsat
7.3 — Hystérésis et immunité au bruit
Le trigger de Schmitt introduit une hystérésis : il n'y a pas un seul seuil de basculement mais deux (VT+ et VT−). Cela rend le circuit insensible aux petites perturbations (bruit).
⚡ Exemple concret

Un trigger de Schmitt avec Vsat = 12V, R1 = 10kΩ, R2 = 100kΩ donne VT+ = +1,2V et VT− = −1,2V. Un signal bruité ne peut pas provoquer de commutations intempestives tant que le bruit est inférieur à 1,2V.

🧠 Quiz — Leçon 7
1. En régime de saturation, quelles valeurs peut prendre la sortie d'un AOP ?
A Une valeur continue quelconque entre −Vcc et +Vcc
B Seulement +Vsat ou −Vsat
C Toujours zéro
D Une valeur dépendant du gain
En régime non linéaire (saturation), la sortie ne peut prendre que deux valeurs : +Vsat ou −Vsat. L'AOP se comporte comme un comparateur binaire.
2. Quel est l'avantage du trigger de Schmitt par rapport à un comparateur simple ?

B Il est immunisé contre le bruit grâce à l'hystérésis
C Il amplifie mieux
D Il consomme moins
L'hystérésis du trigger de Schmitt définit deux seuils de basculement différents (VT+ et VT−), ce qui évite les commutations intempestives dues au bruit sur le signal d'entrée.
3. Un comparateur a V+ = 3V et V− = 5V, avec Vsat = ±12V. Quelle est la sortie ?
A −12V
B +12V
C 0V
D 3V
V+ (3V) < V− (5V), donc Vs = −Vsat = −12V. La règle : si V+ < V−, la sortie sature à −Vsat.

8.1 — Définition et types de filtres
Un filtre est un circuit électronique qui laisse passer certaines fréquences et en atténue d'autres. Un filtre passif est constitué uniquement de composants passifs (R, L, C) sans alimentation externe.

8.2 — Filtre RC passe-bas du 1er ordre
📐 Fréquence de coupure (à −3dB)
fc = 1 / (2π × R × C)
À la fréquence de coupure fc, le gain vaut 1/√2 ≈ 0,707 (-3dB). La pente de coupure d'un filtre du 1er ordre est de −20 dB/décade.
8.3 — Filtre LC (2ème ordre)
Le filtre LC (bobine + condensateur) est un filtre du 2ème ordre, plus sélectif avec une pente de −40 dB/décade.
📐 Fréquence de résonance
f0 = 1 / (2π × √(L × C))
⚡ Exemple concret

Un filtre RC passe-bas avec R = 10kΩ et C = 1µF donne fc = 1/(2π × 10000 × 0,000001) = 15,9 Hz. Il laisse passer les signaux audio basses fréquences et coupe les hautes fréquences.

Le montage Sallen-Key est le filtre actif le plus courant. Il réalise un filtre du 2ème ordre avec deux résistances, deux condensateurs et un AOP monté en suiveur (gain = 1) ou avec amplification.

9.3 — Gabarits de filtres normalisés
Selon leur réponse fréquentielle, les filtres actifs sont classés :
• Butterworth : réponse plate dans la bande passante, pente −40dB/décade (2ème ordre)
• Chebyshev : ondulations dans la bande passante, mais coupure plus nette
• Bessel : réponse en phase linéaire, préserve la forme des signaux temporels
⚡ Exemple concret

En audio, un filtre actif Butterworth passe-bas à 20 kHz protège les tweeter (haut-parleurs aigus) en supprimant les basses fréquences. Il est utilisé dans les filtres de crossover des enceintes acoustiques hi-fi.

10.1 — Définition et condition d'oscillation
Un oscillateur est un circuit électronique qui génère un signal périodique sans signal d'entrée externe. Il utilise une contre-réaction positive pour entretenir les oscillations.
💡 Critère de Barkhausen
Pour qu'un oscillateur oscille de façon entretenue, il faut que :
• Le gain en boucle ouverte soit ≥ 1 : |A × β| ≥ 1
• La phase de la boucle soit un multiple de 360° : ∠Aβ = 0° ou 360°

12.1 — Démarche d'analyse d'un schéma
L'analyse d'un circuit électronique suit une méthode rigoureuse :
Étape 1 — Identifier les blocs fonctionnels : Repérer les grandes parties du schéma (alimentation, entrée, traitement, sortie).

Étape 2 — Identifier les composants : Repérer chaque composant (transistors, AOP, R, C, L) et leur rôle.

Étape 3 — Analyser les liaisons : Suivre le chemin du signal de l'entrée vers la sortie.

Étape 4 — Calculer les grandeurs : Appliquer les lois (Kirchhoff, loi d'Ohm, formules des composants).

Étape 5 — Conclure : Expliquer le fonctionnement global et le rôle du circuit.

12.3 — Exemple d'analyse : Alarme sonore
🔬 Analyse d'un circuit d'alarme

Schéma : Capteur (LDR) → Pont diviseur → AOP comparateur → Transistor NPN → Buzzer

Analyse bloc par bloc :
• LDR + R : pont diviseur de tension. Quand la lumière baisse, VLdR monte.
• AOP comparateur : compare VLDR à Vref. Si VLDR > Vref → sortie = +Vsat.
• Transistor NPN : commandé par la sortie AOP. Quand IB > 0 → transistor sature → buzzer alimenté.
• Buzzer : émet un son quand il reçoit l'alimentation.

Conclusion : Ce circuit est une alarme anti-intrusion : il sonne quand la lumière est coupée (quelqu'un passe devant le capteur).
