Équations, Inéquations et Systèmes linéaires

ÉquationsInéquationsSystèmes
▼
1.1 — Équations du second degré
Une équation du second degré est de la forme ax² + bx + c = 0 avec a ≠ 0. La résolution passe par le calcul du discriminant Δ.
📌 Formule clé — Discriminant
Δ = b² − 4ac
Si Δ > 0 : deux solutions x₁ = (−b − √Δ) / 2a et x₂ = (−b + √Δ) / 2a
Si Δ = 0 : une solution double x₀ = −b / 2a
Si Δ < 0 : pas de solution réelle
1.2 — Inéquations
Une inéquation est une inégalité comportant une inconnue. Attention : multiplier ou diviser par un nombre négatif inverse le sens de l'inégalité.
• Résoudre 2x − 3 > 7 → 2x > 10 → x > 5 → solution : ]5 ; +∞[
• Pour les inéquations du second degré, on utilise le signe du trinôme en étudiant les racines et le signe de a.
💡 Règle du signe
Si a > 0 : ax² + bx + c > 0 pour x < x₁ ou x > x₂ (à l'extérieur des racines).
Si a > 0 : ax² + bx + c < 0 pour x₁ < x < x₂ (entre les racines).
1.3 — Systèmes linéaires
Un système de deux équations à deux inconnues peut être résolu par substitution ou combinaison linéaire (addition). On peut aussi utiliser la méthode de Cramer en classe de Terminale.
🔢 Exemple — Résolution par combinaison

Système : { 2x + y = 5 et x − y = 1 }
Addition : 3x = 6 → x = 2
Substitution : 2 − y = 1 → y = 1
Solution : (2 ; 1)

🧠 Quiz — Chapitre 1
1. Quel est le discriminant de x² − 5x + 6 = 0 ?
A 1
B 1 (Δ = 25 − 24 = 1)
C −24
D 0
Δ = b² − 4ac = (−5)² − 4×1×6 = 25 − 24 = 1 > 0 → deux solutions réelles distinctes x₁=2 et x₂=3.
2. Résoudre −2x + 4 ≥ 0. Quelle est la solution ?
A x ≥ 2
B x ≤ 2
C x ≥ −2
D x ≤ −2
−2x ≥ −4 → on divise par −2 (on inverse) → x ≤ 2. Solution : ]−∞ ; 2].
3. Un système a Δ = 0. Combien de solutions admet l'équation associée ?
A Aucune
B Deux solutions distinctes
C Une solution double
D Une infinité
Δ = 0 → une seule solution appelée racine double : x₀ = −b / 2a.
✏️ Exercices — Chapitre 1
Exercice 1.1
Résoudre dans ℝ l'équation 2x² − 7x + 3 = 0, puis indiquer l'ensemble solution de 2x² − 7x + 3 ≤ 0.
Calcul du discriminant : Δ = (−7)² − 4×2×3 = 49 − 24 = 25 > 0
Racines : x₁ = (7 − 5) / 4 = 1/2 et x₂ = (7 + 5) / 4 = 3
Inéquation ≤ 0 : Comme a = 2 > 0, le trinôme est négatif entre les racines.
Solution : [1/2 ; 3]
Exercice 1.2
Résoudre le système : { 3x + 2y = 12 et 5x − y = 1 } par la méthode de combinaison.
On multiplie la 2ème équation par 2 : 10x − 2y = 2
Addition avec la 1ère : 13x = 14 → x = 14/13
On substitue : y = 5×(14/13) − 1 = 70/13 − 13/13 = 57/13
Solution : (14/13 ; 57/13)

Calculs barycentriques

BarycentreVecteursGéométrie
▼
2.1 — Définition du barycentre
Le barycentre G de points A₁, A₂, ..., Aₙ affectés de coefficients (masses) α₁, α₂, ..., αₙ est le point tel que :
📌 Relation fondamentale
α₁·GA₁ + α₂·GA₂ + ... + αₙ·GAₙ = 0⃗
avec α₁ + α₂ + ... + αₙ ≠ 0
💡 Cas particulier — Milieu
Le milieu I de [AB] est le barycentre de (A, 1) et (B, 1). On a IA⃗ + IB⃗ = 0⃗.
2.2 — Propriétés et associativité
Le barycentre possède une propriété d'associativité : on peut calculer un barycentre global en regroupant les points.
Si G₁ est le barycentre de (A, α) et (B, β), alors le barycentre de (A, α), (B, β) et (C, γ) est aussi le barycentre de (G₁, α+β) et (C, γ).
📐 Exemple — Barycentre de 3 points

G = barycentre de A(1;0), B(3;2), C(2;4) avec coefficients 2, 1, 1.
Σα = 2+1+1 = 4
x_G = (2×1 + 1×3 + 1×2) / 4 = 7/4
y_G = (2×0 + 1×2 + 1×4) / 4 = 6/4 = 3/2

2.3 — Applications géométriques
• Le centre de gravité d'un triangle ABC est le barycentre de (A,1), (B,1), (C,1).
• Ses coordonnées sont : G = ((xA+xB+xC)/3 ; (yA+yB+yC)/3)
• Le barycentre d'un ensemble de points est invariant par translation.
🧠 Quiz — Chapitre 2
1. Le barycentre de (A, 1) et (B, 1) est :
A Le point A
B Le milieu de [AB]
C Le point B
D Un point extérieur à [AB]
Avec des coefficients égaux, le barycentre est le milieu du segment. GA⃗ + GB⃗ = 0⃗ ↔ G est le milieu de [AB].
2. Pour que le barycentre existe, la somme des coefficients doit être :
A Égale à 1
B Positive
C Différente de 0
D Égale à 0
La somme des coefficients doit être ≠ 0, sinon le barycentre n'est pas défini (point à l'infini).
3. Les coordonnées du centre de gravité G du triangle A(0,0), B(6,0), C(0,6) sont :
A (3 ; 3)
B (2 ; 2)
C (6 ; 6)
D (1 ; 1)
G = ((0+6+0)/3 ; (0+0+6)/3) = (6/3 ; 6/3) = (2 ; 2).
✏️ Exercices — Chapitre 2
Exercice 2.1
Calculer le barycentre G de A(−1 ; 2), B(3 ; 0) et C(1 ; 4) affectés des coefficients respectifs 3, 1 et 2.
Somme des coefficients : 3 + 1 + 2 = 6
x_G = (3×(−1) + 1×3 + 2×1) / 6 = (−3 + 3 + 2) / 6 = 2/6 = 1/3
y_G = (3×2 + 1×0 + 2×4) / 6 = (6 + 0 + 8) / 6 = 14/6 = 7/3
G = (1/3 ; 7/3)

Angles orientés et Trigonométrie

Cercle trigonométriqueRadiansFormules
▼
3.1 — Cercle trigonométrique & angles orientés
Le cercle trigonométrique est le cercle de centre O et de rayon 1. On y mesure les angles en radians dans le sens direct (sens antihoraire).
• 180° = π rad  |  90° = π/2 rad  |  60° = π/3 rad  |  45° = π/4 rad  |  30° = π/6 rad
📌 Valeurs remarquables
cos(0) = 1, sin(0) = 0
cos(π/6) = √3/2, sin(π/6) = 1/2
cos(π/4) = √2/2, sin(π/4) = √2/2
cos(π/3) = 1/2, sin(π/3) = √3/2
cos(π/2) = 0, sin(π/2) = 1
3.2 — Formules d'addition et de duplication
📌 Formules d'addition
cos(a + b) = cos a · cos b − sin a · sin b
sin(a + b) = sin a · cos b + cos a · sin b
cos(2a) = cos²a − sin²a = 1 − 2sin²a = 2cos²a − 1
sin(2a) = 2 sin a · cos a
3.3 — Équations trigonométriques
Résolution des équations du type cos(x) = k ou sin(x) = k pour k ∈ [−1 ; 1].
• cos(x) = cos(a) → x = a + 2kπ ou x = −a + 2kπ (k ∈ ℤ)
• sin(x) = sin(a) → x = a + 2kπ ou x = π − a + 2kπ (k ∈ ℤ)
🧠 Quiz — Chapitre 3
1. Quelle est la valeur de sin(π/3) ?
A 1/2
B √3/2
C √2/2
D 1
sin(π/3) = sin(60°) = √3/2 ≈ 0,866. À retenir absolument dans le tableau des valeurs remarquables.
2. Combien de radians correspond à 270° ?
A π
B 2π
C 3π/2
D π/2
270° × (π/180°) = 270π/180 = 3π/2 radians.
3. Quelle est la formule de cos(2a) ?
A 2cos(a)
B 2sin(a)cos(a)
C cos²a − sin²a
D sin²a − cos²a
cos(2a) = cos²a − sin²a. On peut aussi écrire 1 − 2sin²a ou 2cos²a − 1 (formes équivalentes).
✏️ Exercices — Chapitre 3
Exercice 3.1
Résoudre dans ℝ l'équation : sin(x) = √3/2
sin(x) = √3/2 → sin(x) = sin(π/3)
Solution :
x = π/3 + 2kπ  ou  x = π − π/3 + 2kπ = 2π/3 + 2kπ  (k ∈ ℤ)

Nombres complexes

Forme algébriqueModuleArgument
▼
4.1 — Définition et forme algébrique
Un nombre complexe est un nombre de la forme z = a + bi, où a et b sont des réels et i est l'imaginaire pur tel que i² = −1.
• a = Re(z) : partie réelle  |  b = Im(z) : partie imaginaire
• Le conjugué de z = a + bi est z̄ = a − bi
📌 Propriété fondamentale
i² = −1  |  i³ = −i  |  i⁴ = 1
z × z̄ = a² + b² = |z|²
4.2 — Module et argument — Forme trigonométrique
Le module de z = a + bi est |z| = √(a² + b²).
L'argument est l'angle θ tel que cos θ = a/|z| et sin θ = b/|z|.
Forme trigonométrique : z = |z|(cos θ + i sin θ)
📌 Formule d'Euler
z = |z| · e^(iθ) = r·e^(iθ)
Multiplication : |z₁z₂| = |z₁|·|z₂|  et  arg(z₁z₂) = arg(z₁) + arg(z₂)
🧠 Quiz — Chapitre 4
1. Quelle est la valeur de i² ?
A 1
B −1
C i
D 0
Par définition de l'unité imaginaire, i² = −1. C'est la base de tout le chapitre sur les complexes.
2. Le module de z = 3 + 4i est :
A 7
B 5
C 25
D 1
|z| = √(3² + 4²) = √(9 + 16) = √25 = 5. C'est un triplet pythagoricien classique (3, 4, 5).
3. Le conjugué de z = 2 − 3i est :
A −2 + 3i
B 2 + 3i
C −2 − 3i
D 3 − 2i
Le conjugué s'obtient en changeant le signe de la partie imaginaire : z̄ = 2 + 3i.
✏️ Exercices — Chapitre 4
Exercice 4.1
Écrire z = (1 + i)² sous forme algébrique, puis calculer son module et son argument.
Forme algébrique : (1+i)² = 1 + 2i + i² = 1 + 2i − 1 = 2i
Module : |2i| = 2
Argument : arg(2i) = π/2 (le nombre est pur imaginaire positif)
Forme trigonométrique : z = 2(cos(π/2) + i·sin(π/2))

Statistiques

MoyenneVarianceÉcart-type
▼
5.1 — Indicateurs de position
📌 Moyenne et médiane
Moyenne : x̄ = (Σ nᵢ·xᵢ) / N  (N = effectif total)
Médiane : valeur qui partage la série en deux moitiés égales
Mode : valeur de plus grande fréquence
5.2 — Indicateurs de dispersion
📌 Variance et écart-type
Variance : V = (Σ nᵢ·(xᵢ − x̄)²) / N = (Σ nᵢ·xᵢ²)/N − x̄²
Écart-type : σ = √V
Étendue : max − min
💡 Interprétation
Un écart-type faible indique que les valeurs sont concentrées autour de la moyenne. Un écart-type élevé indique une forte dispersion.
5.3 — Statistiques à deux variables (Corrélation linéaire)
On étudie la relation entre deux caractères X et Y. La droite de régression de y en x est : y = ax + b.
• Coefficient de corrélation linéaire r ∈ [−1 ; 1]
• |r| proche de 1 : forte corrélation linéaire
🧠 Quiz — Chapitre 5
1. La série : 2, 4, 4, 6, 8, 10. Quelle est la moyenne ?
A 5
B 5,67
C 4
D 6
x̄ = (2+4+4+6+8+10)/6 = 34/6 ≈ 5,67.
2. L'écart-type est la racine carrée de :
A La moyenne
B La variance
C La médiane
D L'étendue
σ = √V où V est la variance. L'écart-type est l'indicateur de dispersion le plus utilisé.
3. Un coefficient de corrélation r = −0,95 indique :
A Pas de corrélation
B Forte corrélation positive
C Forte corrélation négative
D Corrélation nulle
|r| = 0,95 proche de 1 → forte corrélation. Le signe négatif indique que quand X augmente, Y diminue.
✏️ Exercices — Chapitre 5
Exercice 5.1
Calculer la variance et l'écart-type de la série : 3, 5, 7, 9, 11.
Moyenne : x̄ = (3+5+7+9+11)/5 = 35/5 = 7
Variance : V = [(3−7)²+(5−7)²+(7−7)²+(9−7)²+(11−7)²]/5
= [16+4+0+4+16]/5 = 40/5 = 8
Écart-type : σ = √8 = 2√2 ≈ 2,83

Généralités sur les fonctions

ParitéMonotonieComposition
▼
6.1 — Rappels et vocabulaire
Une fonction f associe à tout x du domaine de définition Df une unique valeur f(x). On note f : Df → ℝ.
• Image de a par f : f(a)
• Antécédent de b par f : tout x tel que f(x) = b
6.2 — Parité, imparité, périodicité
• f est paire si ∀x ∈ Df : f(−x) = f(x) → symétrie par rapport à l'axe des ordonnées.
• f est impaire si ∀x ∈ Df : f(−x) = −f(x) → symétrie par rapport à l'origine.
• f est périodique de période T si f(x + T) = f(x) pour tout x.
📊 Exemples

• cos est paire (cos(−x) = cos x) | sin est impaire (sin(−x) = −sin x)
• sin et cos sont périodiques de période 2π

6.3 — Monotonie et extrema
f est croissante sur I si : x₁ < x₂ ⟹ f(x₁) < f(x₂).
f est décroissante sur I si : x₁ < x₂ ⟹ f(x₁) > f(x₂).
Un extremum (maximum ou minimum) est un point où f change de monotonie.
🧠 Quiz — Chapitre 6
1. f(x) = x² est une fonction :
A Paire
B Impaire
C Ni paire ni impaire
D Périodique
f(−x) = (−x)² = x² = f(x) → f est paire. Sa courbe est symétrique par rapport à l'axe des ordonnées.
2. Quelle est la période de la fonction sinus ?
A π
B 2π
C π/2
D 4π
sin(x + 2π) = sin(x). La période de sin et cos est 2π.
3. f est croissante sur [a, b] signifie que pour x₁ < x₂ on a :
A f(x₁) < f(x₂)
B f(x₁) > f(x₂)
C f(x₁) = f(x₂)
D f(x₁) ≤ f(x₂)
Croissante signifie que la fonction "monte" : plus x est grand, plus f(x) est grand (strictement).
✏️ Exercices — Chapitre 6
Exercice 6.1
Déterminer si f(x) = x³ − x est paire, impaire ou ni l'une ni l'autre. Préciser la symétrie de sa courbe.
f(−x) = (−x)³ − (−x) = −x³ + x = −(x³ − x) = −f(x)
Conclusion : f est impaire.
Sa courbe est symétrique par rapport à l'origine O.

Limites et continuité

LimitesAsymptotesTVI
▼
7.1 — Notion de limite
La limite de f(x) quand x → a est ℓ si f(x) se rapproche de ℓ lorsque x se rapproche de a.
On note : lim(x→a) f(x) = ℓ
• Formes indéterminées : ∞ − ∞, 0 × ∞, ∞/∞, 0/0 → nécessitent une levée d'indétermination.
🔢 Exemples de limites usuelles

• lim(x→+∞) 1/x = 0
• lim(x→0⁺) ln(x) = −∞
• lim(x→+∞) eˣ = +∞
• lim(x→+∞) xⁿ = +∞ (n ∈ ℕ*)

7.2 — Asymptotes
• Asymptote horizontale y = ℓ : si lim(x→±∞) f(x) = ℓ
• Asymptote verticale x = a : si lim(x→a) f(x) = ±∞
• Asymptote oblique y = ax + b : si lim(x→±∞) [f(x) − ax − b] = 0
7.3 — Théorème des valeurs intermédiaires (TVI)
📌 Théorème (TVI)
Si f est continue sur [a ; b] et si f(a) et f(b) sont de signes contraires, alors il existe au moins un c ∈ ]a ; b[ tel que f(c) = 0.
🧠 Quiz — Chapitre 7
1. lim(x→+∞) (3x² + 2x − 1) / x² est égal à :
A +∞
B 3
C 0
D 2
On divise par x² : (3 + 2/x − 1/x²) → 3 quand x → +∞. C'est la méthode du terme dominant.
2. f(x) = 1/(x−2). Quelle est l'asymptote verticale ?
A y = 2
B x = 2
C x = 0
D y = 0
f(x) → ±∞ quand x → 2, donc x = 2 est l'asymptote verticale. y = 0 est l'asymptote horizontale.
3. Le TVI garantit l'existence d'une solution à condition que f soit :
A Dérivable
B Continue
C Croissante
D Paire
Le TVI requiert uniquement la continuité sur [a,b]. Une fonction dérivable est continue, mais la réciproque est fausse.
✏️ Exercices — Chapitre 7
Exercice 7.1
Montrer que l'équation x³ − 3x + 1 = 0 admet au moins une solution dans [1 ; 2].
Soit f(x) = x³ − 3x + 1. f est continue sur ℝ (polynôme).
f(1) = 1 − 3 + 1 = −1 < 0
f(2) = 8 − 6 + 1 = 3 > 0
f(1) et f(2) sont de signes opposés → par le TVI, il existe c ∈ ]1 ; 2[ tel que f(c) = 0. ✓

Dérivation

DérivéeTangenteRègles
▼
8.1 — Définition et interprétation
La dérivée de f en a est : f'(a) = lim(h→0) [f(a+h) − f(a)] / h
Interprétation : f'(a) est le coefficient directeur de la tangente à la courbe en A(a, f(a)).
Équation de la tangente en A : y = f'(a)(x − a) + f(a)
8.2 — Tableau des dérivées usuelles

k (constante)0
xⁿn·xⁿ⁻¹
√x1 / (2√x)
1/x−1/x²
eˣeˣ
ln(x)1/x
sin(x)cos(x)
cos(x)−sin(x)
8.3 — Règles de calcul
📌 Règles opératoires
(u + v)' = u' + v'
(k·u)' = k·u'
(u·v)' = u'·v + u·v'
(u/v)' = (u'·v − u·v') / v²
(f∘g)'(x) = g'(x) · f'(g(x))  [dérivée composée]
🧠 Quiz — Chapitre 8
1. La dérivée de f(x) = x⁴ est :
A x³
B 4x³
C 4x⁴
D x⁵/5
Règle : (xⁿ)' = n·xⁿ⁻¹. Donc (x⁴)' = 4·x³.
2. La dérivée de f(x) = eˣ est :
A xeˣ⁻¹
B eˣ
C x·eˣ
D ln(x)
La fonction exponentielle est sa propre dérivée : (eˣ)' = eˣ. C'est une propriété remarquable.
3. L'équation de la tangente à f(x) = x² en x = 1 est :
A y = x
B y = 2x − 1
C y = 2x + 1
D y = x + 1
f'(x) = 2x → f'(1) = 2. Point A(1, 1). Tangente : y = 2(x−1) + 1 = 2x − 1.
✏️ Exercices — Chapitre 8
Exercice 8.1
Calculer la dérivée de f(x) = (2x + 1)·eˣ, puis écrire l'équation de la tangente au point d'abscisse x = 0.
Produit u·v avec u = 2x+1 et v = eˣ
u' = 2, v' = eˣ
f'(x) = 2·eˣ + (2x+1)·eˣ = eˣ(2x + 3)
En x=0 : f(0) = 1·1 = 1 et f'(0) = 1·3 = 3
Tangente : y = 3(x − 0) + 1 = 3x + 1

Étude des fonctions

Tableau de variationExtremaGraphe
▼
9.1 — Méthode d'étude complète
Pour étudier une fonction, on suit ces étapes :
1. Déterminer le domaine de définition Df
2. Étudier la parité ou la périodicité
3. Calculer les limites aux bornes
4. Calculer f'(x) et étudier son signe
5. Dresser le tableau de variation
6. Identifier les extrema et tracer la courbe
9.2 — Lien dérivée–monotonie
📌 Règle fondamentale
• Si f'(x) > 0 sur I : f est strictement croissante sur I
• Si f'(x) < 0 sur I : f est strictement décroissante sur I
• Si f'(a) = 0 avec changement de signe : f admet un extremum local en a
9.3 — Convexité et point d'inflexion
• f est convexe sur I si f''(x) ≥ 0 sur I (courbe au-dessus de ses tangentes)
• f est concave sur I si f''(x) ≤ 0 sur I
• Un point d'inflexion est un point où f'' change de signe.
🧠 Quiz — Chapitre 9
1. Si f'(x) < 0 sur ]a ; b[, alors f est :
A Croissante
B Décroissante
C Constante
D Nulle
Une dérivée négative signifie que la fonction "descend" : elle est strictement décroissante.
2. f(x) = −x² + 4x. En quel point f admet-elle son maximum ?
A x = 0
B x = 4
C x = 2
D x = −2
f'(x) = −2x + 4 = 0 → x = 2. Comme f'' = −2 < 0, c'est bien un maximum. f(2) = −4 + 8 = 4.
3. Un point d'inflexion se trouve là où :
A f'(x) = 0
B f''(x) change de signe
C f(x) = 0
D f est continue
Un point d'inflexion est un point de la courbe où la convexité change, c'est-à-dire où f'' change de signe.
✏️ Exercices — Chapitre 9
Exercice 9.1
Étudier les variations de f(x) = x³ − 3x + 2 sur ℝ et dresser son tableau de variation.
f'(x) = 3x² − 3 = 3(x² − 1) = 3(x−1)(x+1)
f'(x) = 0 pour x = −1 et x = 1
• f'(x) > 0 sur ]−∞ ; −1[ et ]1 ; +∞[ → croissante
• f'(x) < 0 sur ]−1 ; 1[ → décroissante
Extrema : f(−1) = −1 + 3 + 2 = 4 (maximum local)
f(1) = 1 − 3 + 2 = 0 (minimum local)

Suites numériques

ArithmétiqueGéométriqueConvergence
▼
10.1 — Suites arithmétiques
Une suite (uₙ) est arithmétique de raison r si : uₙ₊₁ = uₙ + r.
📌 Formules — Suite arithmétique
Terme général : uₙ = u₀ + n·r
Somme des n+1 premiers termes : Sₙ = (n+1)·(u₀ + uₙ) / 2
10.2 — Suites géométriques
Une suite (uₙ) est géométrique de raison q si : uₙ₊₁ = q·uₙ (avec q ≠ 0 et u₀ ≠ 0).
📌 Formules — Suite géométrique
Terme général : uₙ = u₀ · qⁿ
Somme : Sₙ = u₀·(1 − qⁿ⁺¹) / (1 − q)  si q ≠ 1
10.3 — Convergence et limites
• Suite arithmétique : diverge vers ±∞ si r ≠ 0
• Suite géométrique : converge vers 0 si |q| < 1 ; diverge si |q| > 1
• Une suite croissante et majorée est convergente (théorème de la limite monotone)

uₙ = u₀ + n·r → u₅ = 2 + 5×3 = 2 + 15 = 17.

11.1 — Repère de l'espace
Dans l'espace, un point M est repéré par ses coordonnées (x ; y ; z) dans un repère orthonormé (O ; i⃗, j⃗, k⃗).
Distance entre A(x₁,y₁,z₁) et B(x₂,y₂,z₂) :
AB = √[(x₂−x₁)² + (y₂−y₁)² + (z₂−z₁)²]
11.2 — Vecteurs et produit scalaire dans l'espace
📌 Produit scalaire 3D
u⃗·v⃗ = x₁x₂ + y₁y₂ + z₁z₂
u⃗·v⃗ = |u⃗|·|v⃗|·cos(θ)
u⃗ ⊥ v⃗ ↔ u⃗·v⃗ = 0
11.3 — Plans et droites dans l'espace
• Équation d'un plan : ax + by + cz + d = 0 où (a, b, c) est un vecteur normal au plan.
• Droite paramétrée : { x = x₀ + at ; y = y₀ + bt ; z = z₀ + ct } (t ∈ ℝ)
• Deux plans sont parallèles si leurs vecteurs normaux sont colinéaires.
• Deux plans sont perpendiculaires si leurs vecteurs normaux sont orthogonaux.

12.1 — Isométries du plan
Une isométrie est une transformation qui conserve les distances. Les isométries du plan sont :
• La translation de vecteur u⃗
• La rotation de centre O et d'angle α
• La réflexion (symétrie axiale) par rapport à une droite d
📌 Translation de vecteur u⃗(a,b)
M(x,y) → M'(x+a, y+b)
12.2 — Rotation
La rotation de centre O et d'angle θ transforme M(x,y) en M'(x',y') avec :
x' = x·cos θ − y·sin θ
y' = x·sin θ + y·cos θ
🔄 Exemple — Rotation de 90° autour de O

M(3, 1) → M'(−1, 3)
x' = 3·cos90° − 1·sin90° = 0 − 1 = −1
y' = 3·sin90° + 1·cos90° = 3 + 0 = 3

12.3 — Homothétie
L'homothétie de centre O et de rapport k transforme M en M' tel que OM'⃗ = k·OM⃗.
• Si k > 0 : M' du même côté que M par rapport à O
• Si k < 0 : M' du côté opposé
• |k| = 1 : isométrie (symétrie centrale si k = −1)
🧠 Quiz — Chapitre 12

A'(1+3 ; 4+(−2)) = A'(4 ; 2).

Par définition, une isométrie conserve les distances. Elle conserve aussi les angles et les aires en conséquence.

Quand k = −1, OM'⃗ = −OM⃗, ce qui signifie que O est le milieu de [MM'] → symétrie centrale de centre O.
